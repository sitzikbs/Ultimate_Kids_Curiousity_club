# Background Music

This directory contains royalty-free background music tracks for podcast episodes.

## Available Tracks

Add your background music files here. Recommended formats:
- MP3 (192 kbps or higher)
- WAV (for highest quality)

## Track Guidelines

- Loopable (seamless transitions)
- Upbeat and positive tone
- Educational-friendly (not distracting)
- 120-140 BPM
- Instrumental only (no vocals)

## License

Ensure all music tracks used are either:
- Royalty-free
- Licensed for podcast/commercial use
- Original compositions

## Sample Track Names

- `background_loop_curious.mp3` - Main theme
- `background_loop_gentle.mp3` - Calm segments
- `background_loop_exciting.mp3` - Action/discovery moments
